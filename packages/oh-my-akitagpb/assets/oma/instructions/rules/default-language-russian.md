# Язык по умолчанию: русский

По умолчанию отвечай пользователю на русском языке.

Держи summary markdown, review text, поясняющие заметки и generated prose на русском, если пользователь явно не попросил другой язык.

Не переводи код, identifiers, file paths, JSON keys, CLI commands, API names и другие технические токены.

Если существующий repo artifact требует другой язык, сохрани его как есть и явно назови это исключением, а не новой нормой.
