# Akita GPB module akita-gpb-core-module
## Назначение сервиса
Сервис akita-gpb-core-module является ядром фреймворка Akita-BDD, отвечающим за основную логику и функциональность. Он обеспечивает взаимодействие между модулями и управляет ключевыми процессами. Сервис относится к домену автоматизации тестирования и поддерживает создание и выполнение тестовых сценариев.
## Описание процесса
В сервисе реализован следующий бизнес-процесс: Управление тестами и окружением:
1. Инициализация и управление тестовым окружением через класс AkitaEnvironment.
2. Управление сценариями тестов с помощью AkitaScenario, включая создание шагов и передачу данных между ними.
3. Интеграция с модулями API, UI, DataBase и другими для выполнения комплексных сценариев.
4. Конфигурация и управление ресурсами тестового окружения.

## Список шагов модуля
Модуль предоставляет следующие шаги для использования в тестах:
```
@И("^сохранено значение \"(.*)\" из property файла в переменную \"(.*)\"$")
@И("^установлено значение переменной \"(.*)\" равным \"(.*)\"$")
@Когда("^выполнено ожидание в течение (\\d+) (?:секунд|секунды)")
@И("^шаблон \"(.*)\" заполнен данными из таблицы и сохранён в переменную \"(.*)\"$")
@Тогда("^значения в переменных \"(.*)\" и \"(.*)\" совпадают$")
@Тогда("^значения в переменных \"(.*)\" и \"(.*)\" не совпадают$")
@И("^удалить файл \"(.*)\" из директории \"(.*)\"$")
@И("^в файле \"(.*)\" содержится текст \"(.*)\"$")
@Тогда("^значения из переменной \"(.*)\" и из property файла \"(.*)\" совпадают$")
@Тогда("^верно, что \"(.*)\"$")
```
