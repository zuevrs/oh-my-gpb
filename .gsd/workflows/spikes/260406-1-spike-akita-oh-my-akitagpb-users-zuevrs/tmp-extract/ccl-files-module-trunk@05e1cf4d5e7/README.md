# ccl-files-module
## Назначение сервиса
Сервис ccl-files-module предоставляет шаги для работы с файлами и строками различных форматов, включая .pdf, .png, .docx, .xlsx, а также текстовые форматы .xml, .json и .html. Он поддерживает операции с файлами и строками, такие как чтение, преобразование и валидация. Сервис относится к домену обработки файлов в автоматизации тестирования.
## Описание процесса
В сервисе реализован следующий бизнес-процесс: Обработка файлов и данных:
1. Сохранение содержимого файлов в переменные и создание рандомных файлов.
2. Работа с .docx (подстановка переменных, сравнение содержимого).
3. Управление .xlsx (чтение ячеек, проверка условий).
4. Извлечение данных из .html (по XPath).
5. Сравнение и сохранение .png-изображений.
6. Обработка .json (JsonPath, сравнение объектов).
7. Анализ .pdf (поиск текста, таблиц).
8. Извлечение данных из .xml (тэги, атрибуты).

## Требования
Данный модуль зависит от модуля ccl-helpers-module этого же проекта. <br>
Но эта зависимость добавляется посредством собранного jar-файла, а не ссылкой на исходники модуля.<br>
Соответственно, для успешной сборки данного модуля jar-архив зависимого модуля должен быть предварительно опубликован в репозитории (локальном или
публичном).<br>
