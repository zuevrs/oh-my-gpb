# ccl-additional-steps-module
## Назначение сервиса
Сервис ccl-additional-steps-module предоставляет общие шаги для работы со строками, числами, датами и массивами, а также шаги для выполнения действий в цикле до достижения условия. Модуль не содержит основной логики и служит для поддержки дополнительных функций в фреймворке. Относится к домену вспомогательных инструментов автоматизации.
## Описание процесса
В сервисе реализован следующий бизнес-процесс: Обработка данных и условий:
1. Сохранение и обработка значений (строки, числа, даты) в переменные сценария.
2. Выполнение вычислений и преобразований (регулярные выражения, форматы, Base64-декодирование).
3. Работа с массивами (сравнение, сохранение).
4. Циклическое выполнение запросов до достижения ожидаемого результата.

## Список шагов модуля
Общие шаги:
```
@И("^значение \"(.*)\" сохраняем в переменную сценария \"(.*)\"$")
@И("^значение \"(.*)\" добавляем к списку в переменной сценария \"(.*)\"$")
@И("^сохраняем в переменные сценария значения из таблицы:?$")
@И("^к переменной \"(.+)\" применить регулярное выражение \"(.+)\" и сохранить результат в переменную \"(.+)\"$")
@И("^вычислить даты, сохранив результат в переменные из таблицы:?$")
@И("^для дат выполняются условия из таблицы:?$")
@И("^выполнить выражение \"(.+)\" и сохранить его результат в переменную \"(.+)\"$")
@И("^сохранить число \"(.+)\" прописью( с большой буквы| в верхнем регистре)? в формате \"(.+)\" в переменную \"(.+)\"$")
@И("^сохранить сумму \"(.+)\" в рублях прописью( с большой буквы| в верхнем регистре)? в переменную \"(.+)\"")
@И("^сохранить число процентов \"(.+)\" прописью( с большой буквы| в верхнем регистре)? в переменную \"(.+)\"")
@И("^сохранить число \"(.+)\" в формате \"(.+)\" в переменную \"(.+)\"")
@И("^из ответа \"(.*)\" значение тела сохранено в виде массива байт в переменную \"(.*)\"$")
@И("^декодировать Base64-строку \"(.*)\" и сохранить полученный массив байт в переменную \"(.*)\"$")
@И("^выполнено ожидание в течение \"(.*)\" (?:секунд|секунды)")
@И("^для строки \"(.*)\" выполняются условия из таблицы:?")
```
Шаги для работы с массивами:
```
@И("^массив \"(.*)\" совпадает с массивом \"(.*)\", тип элементов (строка|число|дата)$")
```
Шаги с выполнением действий в цикле до достижения ожидаемого условия:
```
@И("^выполнен (GET|POST|PUT|DELETE|PATCH) запрос на URL \"(.*)\" с headers и parameters из таблицы. В ответе ожидаем что jsonPath \"(.*)\" равен \"(.*)\"$")
```

## Требования
Данный модуль зависит от модулей ccl-helpers-module и akita-gpb-api-module этого же проекта. <br>
Но эти зависимости добавляются посредством собранных jar-файлов, а не ссылкой на исходники модулей.<br>
Соответственно, для успешной сборки данного модуля jar-архивы зависимых модулей должен быть предварительно опубликованы в репозитории (локальном или
публичном).
